# dbt Star Schema

## Описание задания

Переписать построение модели "звезда" из задания Недели 4 (Проектирование данных: от OLTP к OLAP) с использованием фреймворка DBT

---

## Источник данных

Для воспроизводимости окружения используется `docker-compose`, который
разворачивает PostgreSQL с OLTP-данными. Таблицы и начальные данные
создаются при старте контейнера. Аналитическая витрина строится поверх этих
данных с использованием dbt.

## Общая логика работы

Исходные данные разворачиваются в PostgreSQL с использованием `docker-compose`. 
Для таблиц клиентов и продуктов фиксируется момент изменения данных через поле `updated_at`. 
История изменений измерений реализуется с помощью dbt snapshots. 
На основе сохранённой истории и таблицы продаж dbt формирует аналитическую модель данных типа «звезда», 
создавая итоговые таблицы измерений и фактов в схеме `analytics`.
